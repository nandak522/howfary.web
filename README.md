howfary.web
========

Pyramid based Web-Frontend for Howfary


Getting Started
---------------

- cd $PROJECT\_HOME

- $venv/bin/python setup.py develop

- $venv/bin/initialize_howfary.web_db development.ini

- $venv/bin/pserve development.ini
